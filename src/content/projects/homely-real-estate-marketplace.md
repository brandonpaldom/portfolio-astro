Homely es una plataforma para encontrar propiedades inmobiliarias. En este sitio podrás encontrar propiedad de tus sueños de forma fácil. Fue desarrollada con Next.js y Cloudinary.

[Visitar Homely](https://homely-marketplace.vercel.app/) • [Ver en GitHub](https://github.com/brandonpaldom/homely)

## Características

- Búsqueda de propiedades por ubicación, precio y características
- Visualización detallada de propiedades con imágenes y descripciones
- Contacto directo con los agentes inmobiliarios
- Guardado de propiedades favoritas
- Interfaz de usuario intuitiva y fácil de usar

## Tecnologías utilizadas

- Next.js para el frontend y routing
- Cloudinary para la gestión de imágenes
- API personalizada para la gestión de propiedades
- Diseño responsivo para dispositivos móviles y de escritorio
