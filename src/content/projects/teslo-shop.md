Teslo Shop es una tienda en línea moderna de ropa inspirada en Tesla, desarrollada con tecnologías de vanguardia como Next.js 15 (con Turbopack), React 19, TypeScript 5 y Tailwind CSS.

[Visitar Teslo Shop](https://teslo-shop-bp.vercel.app/) • [Ver en GitHub](https://github.com/brandonpaldom/teslo-shop)

## Características

- Catálogo de productos con filtros por categoría y tamaño
- Carrito de compras
- Proceso de pago con PayPal
- Panel de administración para gestionar productos, órdenes y usuarios
- Autenticación de usuarios
- Gestión de órdenes

## Tecnologías

El proyecto utiliza Prisma ORM conectado a Neon Serverless PostgreSQL para una base de datos escalable y de alto rendimiento, complementado con Cloudinary para la gestión de imágenes.
