Cystore es una tienda en línea de productos tecnológicos, desarrollada con Next.js, Tailwind CSS, Firebase y Clerk. Ofrece una experiencia de compra en línea moderna y completa para una amplia gama de productos tecnológicos, con un diseño limpio y elegante.

[Visitar Cystore](https://cystore.vercel.app/) • [Ver en GitHub](https://github.com/brandonpaldom/cystore)

## Características

- Catálogo de productos con filtros por categoría
- Carrito de compras
- Autenticación de usuarios con Clerk
- Base de datos en Firebase
- Diseño responsivo con Tailwind CSS
- Interfaz de usuario moderna y elegante

## Tecnologías utilizadas

- Next.js para el frontend y routing
- Tailwind CSS para el diseño
- Firebase para la base de datos
- Clerk para la autenticación de usuarios
