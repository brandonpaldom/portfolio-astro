Landing pages para diversas promociones inmobiliarias. Los sitios muestran los inmuebles y ofrecen información sobre sus características y servicios, para que los compradores potenciales puedan obtener más información e imaginarse viviendo allí.

## Tecnologías utilizadas

- HTML, CSS y JavaScript para el frontend
- WordPress como CMS
- Elementor para la construcción de páginas
- Formularios de contacto integrados con CRM

## Proyectos incluidos

### Veramonte Residencial

Landing page para un desarrollo residencial de lujo, con información detallada sobre las propiedades, amenidades y ubicación.

![Veramonte Residencial](https://res.cloudinary.com/dlomynswh/image/upload/v1678916575/portfolio/projects/real-estate-landing-pages-1.jpg)

### City Blue Santa Fe

Sitio web para un desarrollo de apartamentos en la zona de Santa Fe, con visualizaciones 3D y detalles sobre los diferentes modelos disponibles.

![City Blue Santa Fe](https://res.cloudinary.com/dlomynswh/image/upload/v1678916577/portfolio/projects/real-estate-landing-pages-2.jpg)

### Mareazul Real Estate

Plataforma para la promoción de propiedades en la Riviera Maya, con enfoque en compradores internacionales.

![Mareazul Real Estate](https://res.cloudinary.com/dlomynswh/image/upload/v1678916577/portfolio/projects/real-estate-landing-pages-3.jpg)

### Aura Towers

Landing page para un desarrollo de torres residenciales, con información sobre las diferentes unidades, amenidades y proceso de compra.

![Aura Towers](https://res.cloudinary.com/dlomynswh/image/upload/v1678916573/portfolio/projects/real-estate-landing-pages-4.jpg)
